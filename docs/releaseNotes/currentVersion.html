ZCSPM Release 3.06.0 Enhancements and Fixes

New features and enhancements

  • Custom Security Policies for AWS:

    ZCSPM now allows you to create custom security policies for your AWS cloud deployment. You can use the AWS Config Aggregator to write queries and build a custom policy, allowing you to protect all the AWS resources in your cloud environment. To learn more, see Creating Custom Security Policies .

  • Asset Inventory :

    • The asset inventory provides an inventory of all the assets deployed on your cloud infrastructure i.e., assets not protected by ZCSPM and assets that are protected by ZCSPM with default security policies. To learn more, see About Asset Inventory .

  • Bulk project onboarding for GCP Organization :

    • You can now onboard multiple GCP projects in a single organization into ZCSPM at once. To learn more, see Onboarding a GCP Organization Account. .

Platform & Stability Improvements

  • K8s agents and AWS Auto Remediation framework have been rebranded to ZCSPM from Cloudneeti.
  • Resolved the following issues:

    • Audit log filter now register special characters on the policy details page.
    • The region filter on Asset Security Dashboard is now fixed
    • Custom polices now show resources on CSBP and Asset Security Dashboard for all resource types.
    • Resources can now be filtered using tags in the Asset Security Dashboard.
    • Total workload and workload peak values for Azure have been corrected.
    • Enabling remediation no longer re-enables disabled policies.
    • GCP firewall policies are now visible on CSBP and Asset Security Dashboard.
    • Custom policy queries now return strings instead of objects.
    • Duplicate security polices are no longer displayed.
    • Asset search functionality enhanced on the Asset Security Dashboard.
    • Aggregated utilization information for Cloud Service Provider and License level are corrected.
    • Total Azure accounts count fixed.
    • M365 security policies now show correctly on the license level.
    • M365 global admin policies now shows the correct count.

Security Policy & Benchmark Updates

  • Added 6 new security policies for Azure.
  • Added 1 new security policy for GCP project.
