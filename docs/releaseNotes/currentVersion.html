February 2020 - Release 2.26.1

  1. User Experience Improvements

    • Asset Security Dashboard Preview: View protected Azure and AWS assets per region.

    • Risk Dashboard: Updates in user interface.

    • Release Notification: Email notification to all users on new release.

    • AWS Remediation: Support for Cloudneeti Remediation Framework different region deployment.

    • Auto Remediation: Added audit logs for Azure and AWS auto remediation.

  2. Platform & Stability Improvements

    • Fixed following bugs

      • Extra categories are visible in the Baseline benchmark list

      • Other resources in a policy are not getting remediated if any one resource fails to remediate due to some reason

      • UI/UX: Msal.idtoken is accepted by cloudneeti api endpoints even after it is expired

      • Resource Not Found in CreatePrivateBenchmarkSummaryActivity (QA, Prod)

      • Exception in RBACCustomerPolicySummaryActivity

      • Missing TTL in AggregatedCISOTrendData AggregatedCISODashboard
